# Gangsters - Система заказа еды

Веб-приложение для заказа еды с интерактивной системой мотивации клиентов.

## Описание

Система управления заказами еды с современным интерфейсом, включающая:

-   **Каталог товаров** с категориями и фильтрацией
-   **Корзину покупок** с интерактивным прогресс-баром
-   **Систему подарков** - мотивация клиентов при достижении определенных сумм заказа
-   **Оформление заказов** с доставкой и самовывозом
-   **Административную панель** для управления товарами и заказами
-   **Интеграции** с Telegram, Yandex Food и другими сервисами

## Технологии

-   **Backend**: Laravel 10, PHP 8
-   **Frontend**: Vue.js 3, Tailwind CSS, GSAP
-   **База данных**: MySQL
-   **Дополнительно**: Pinia (state management), Material Design Icons

## Особенности

-   Адаптивный дизайн для всех устройств
-   Интерактивные анимации и переходы
-   Система уведомлений в реальном времени
-   Многоязычная поддержка
-   Интеграция с платежными системами
